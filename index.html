The Problem

Every parent worries about SIDS and baby safety

  • Sleep Position Risks

    Babies sleeping on their stomach or side face higher SIDS risk, but constant monitoring is exhausting.

  • Parent Exhaustion

    New parents can't watch their baby 24/7, leading to anxiety and sleep deprivation.

  • Understanding Cries

    Decoding why a baby is crying is challenging, especially for first-time parents.

Artificial Intelligence

Powered by 3 ML Models

Advanced machine learning models working together to provide comprehensive safety monitoring.

Sleep Position Detector

Analyzes baby's body posture to detect safe (back) vs unsafe (stomach/side) sleeping positions using pose estimation.

Model RandomForest
Features 142 Engineered
Landmarks 33 Pose Points
Backend MediaPipe + sklearn

Awake/Sleep Detector

Monitors eye region to determine if the baby is awake or asleep using deep learning image classification.

Model MobileNetV2 CNN
Input 64×64×3 Images
Classes Awake / Asleep
Backend TensorFlow + Keras 3

Cry Detection & Analysis

Two-stage system: first detects if baby is crying, then classifies the reason with actionable recommendations.

Stage 1 Binary CNN
Stage 2 6-Class Classifier
Reasons Hungry, Tired, Pain...
Backend librosa + TensorFlow
